このリポジトリには，論文「印刷中」で使用されているサンプルコードが含まれています．
データセットは完全に合成データであり，教育目的のみで生成されています．
実際の患者データは一切含まれていません．

必要条件
- Python 3.10以降
- pandas
- numpy
- statsmodels

このコードでは，以下の内容が示されています．
1. 架空の臨床データの生成
2. 重回帰分析
3. 単回帰分析との比較

推奨手順：
1.Google Colaboratoryでノートブックを開く
2.セルを順番に実行する
3.回帰係数・p値・R²・adjusted R²を確認する
4.サンプルサイズや説明変数を変更して結果の変化を観察する
5.多重共線性や過学習の影響を確認する
